# 傀儡足球 — 2026世界杯

1v1 傀儡物理足球对战。两个傀儡球员在球场上拼抢、跳跃、踢球，物理碰撞制造各种搞笑场面。

## 操作

| 动作 | P1 | P2 |
|------|-----|-----|
| 左移 | A | ← |
| 右移 | D | → |
| 跳跃 | W | ↑ |
| 踢球 | S | ↓ |
